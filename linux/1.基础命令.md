* 进入路径  
```
cd ..    进入上一层路径  
cd xx    进入xx文件夹  
cd /   进入根目录  
```
* 查看目录下的全部文件和文件夹  
` ls`  
* 查看系统版本信息  
`uname -a`
* 查看linux的发行版本   
`lsb_release -a`
* 终止操作(快捷键)  
`ctrl + C`  
* 清空屏幕(快捷键)  
`ctrl + L`
* 查看当前目录  
`pwd`  
* 日期类操作  
```
查看日期 date 
格式化日期 date +%Y-%m-%d
计算少一天 date -d "-1 days" +%Y-%m-%d
计算少一天 date -d "1 days ago" +%Y-%m-%d
设置系统时间 date -s "2019-10-24 8:00"
```
