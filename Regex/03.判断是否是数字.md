#### 判断字符串是否是数字，包含负号和小数点

```java
    public static void main(String[] args) {
        String str = "-2165484213.8";
        Pattern pattern = Pattern.compile("(-?[1-9]\\d*\\.?\\d*)|(0\\.\\d*[1-9])");
        Matcher isNum = pattern.matcher(str);
        if( !isNum.matches() ){
            System.out.println(false);
        }else {
            System.out.println(true);
        }
    }
```
