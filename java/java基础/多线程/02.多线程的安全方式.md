## 多线程的安全问题，在于如果没有使用安全的多线程调用，会出现程序执行一半以后跳到另一个线程，导致出现各种问题，如已经卖掉的票未显示卖掉结果卖了两次。此时只要加入同步关键词就可以解决，同步关键词的方法如下所示。需要注意的是，使用同步代码块时，加的锁是this保证多线程共用一把锁，而同步方法里面默认使用this保证多线程使用同一把锁。
### 同步代码块
```java
public void run(){
  ...
  synchronized(this){
    ...
  }
}
```

### 同步方法
```java
public synchronized void run(){
  ...
  sale();
}
public void sale(){
  ...
}
```
