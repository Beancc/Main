#### 背景是在做人员档案时候需要输入年龄用的是时间控件，然而默认年份选择是当前年份的前十年和后十年，既点开下拉框会显示2009-2029年。（做这个项目的时候是2019年）然而显然不满足年龄，
当时查了一些资料发现方法都不太适合这个项目，最终在官方文档找到了以下方案。下拉框显示的数据是当前日期-60年到当前日期。


```
$('#asmBirth').datepicker({
  changeYear:'true',
  changeMonth:'true',
  yearRange:"c-60:c"
});

```
